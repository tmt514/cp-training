Shortest Repetition

The Task

You are given a string SSS. Find the shortest string XXX such that SSS is a repetition of XXX.

For example, if S=abcabcabcabcS=\texttt{abcabcabcabc}S=abcabcabcabc then your algorithm should output X=abcX=\texttt{abc}X=abc.

Goals​

  • Find an O(nn)O(n \sqrt{n})O(nn​) time algorithm. (Hint: enumerate all possible lengths of XXX.)
  • Find an O(nlog⁡n)O(n \log n)O(nlogn) time algorithm. (Hint: built on above idea, but check each match in O(1)O(1)O(1) time using Hashing.)
  • Find an O(n)O(n)O(n) time algorithm. (Hint: ...)
