Basic String Manipulations

There are three important operations for strings: splitting, appending, and type conversion. Fortunately, for Java and Python we already have libraries that help us accomplish these tasks.

Splitting​

  • Java
  • Python
String input = Scanner.nextLine()
String[] strs = input.trim().split(" ")
strs = input().strip().split()

Appending​

  • Java
  • Python

Java strings are immutable, which means that whenever you attempt to append a character (or other string) to the existing one, you must allocate a new string. Therefore, the naive way to appending nnn characters to a string actually takes O(n2)O(n^2)O(n2) time! To resolve this issue, you can either use ArrayList or StringBuilder.

public class arraylist {
  public static void main(String[] args) {
    ArrayList<String> strings = new ArrayList<String>();
    strings.add("one");
    strings.add("two");
    strings.add("three");
    strings.add("four");
    String all = String.join("+", strings);
    System.out.println(all);
  }
}
